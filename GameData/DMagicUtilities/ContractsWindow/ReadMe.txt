Place the contents of the GameData folder in your KSP/GameData folder:

Kerbal Space Program/GameData/DMagicUtilities/

Contracts Window will not function properly if not placed in the correct location.

Contracts Window has two required dependencies; Contract Parser and Progress Parser. Both are included in the download package and can be found in the same folder as Contracts Window. Both are alos dependencies for CapCom and should not be removed unless both of those addons are removed.


*Blizzy78's Toolbar is supported, but not required
